# TF2_CLASSES

These are predefined classes for TF2. `lib_players.lua` has a `get_tf2_class` function that returns the class of a player. You can also manually check by using `DT_TFPlayer` and `m_iClass` in `source:get_netvar`. 

| TF2 Class    | Class Name |
|--------------|------------|
| TF2_SCOUT    | Scout      |
| TF2_SOLDIER  | Soldier    |
| TF2_PYRO     | Pyro       |
| TF2_DEMOMAN  | Demoman    |
| TF2_HEAVY    | Heavy      |
| TF2_ENGINEER | Engineer   |
| TF2_MEDIC    | Medic      |
| TF2_SNIPER   | Sniper     |
| TF2_SPY      | Spy        |
