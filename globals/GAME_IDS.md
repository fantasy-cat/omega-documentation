# GAME_IDS

Game IDs are automatically assigned when `process:set` recognizes a process name or origin. `GAME_DEADLOCK` and other titles are not supported yet in Omega, yet the constants still exist for later.

| GAME_          | Associated Process                     |
|----------------|----------------------------------------|
| GAME_CS2       | CS2                                    |
| GAME_TF2       | TF2                                    |
| GAME_CSS       | Any title with `hl_linux` or `hl2.exe` |
| GAME_L4D2      | L4D2                                   |
| GAME_DEADLOCK  | Missing logic in Omega at the moment.  |
| GAME_CSGO      | Logic removed in Omega.                |
| GAME_MINECRAFT | Logic removed in Omega.                |
